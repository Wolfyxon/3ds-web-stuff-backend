# Security Policy

## Reporting a Vulnerability
If you've found a security vulnerability, please do not use issues as such things should be kept private.

Please use [Security advisories](https://github.com/Wolfyxon/3ds-web-stuff-backend/security/advisories) to privately report them.

## Example vulnerabilities
- SQL injection
- Public visibility of sensitive data (such as password hashes)
- Bypassing protection
- Changing data you're not supposed to change
