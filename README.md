# 3DS Web Stuff backend
Server side API functions for [3DS Web Stuff](https://wolfyxon.github.io/3ds-web-stuff/)

## Environment variables
- `CONFIG`: Path to the config JSON file
### PostreSQL Database
- `POSTGRES_URL`
- `POSTGRES_URL_NON_POOLING`
- `POSTGRES_URL_NO_SSL`
- `POSTGRES_PRISMA_URL`
- `POSTGRES_USER`
- `POSTGRES_PASSWORD`
- `POSTGRES_HOST`
- `POSTGRES_DATABASE`
